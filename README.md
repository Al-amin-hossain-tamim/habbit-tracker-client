# HabitSpark

### Live Website URL  


##  Features of HabitSpark

- Users can create habits with title, description, category, and image.
- Private routes protect Add Habit, My Habits, and Habit Details pages.
- Users can mark a habit as completed and track streaks + progress.
- Public Habits page allows anyone to browse habits created by all users.
- Firebase Authentication supports Email/Password and Google login.
- Responsive UI built with React, TailwindCSS, and DaisyUI.
